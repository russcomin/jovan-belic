The beggining

My Story

My name is Jovan Belic, date when i saw light for the first
time: 15.1.2001. I live in Zabrezje, small city, near Belgrade
the capital of Serbia. Im not really focused person, i love
doing as much things as i can. I started coding when i was
14, i went to a Game Development Workskop and after that
i really started to think with my own head. For the first few
months i’ve been doing only Game Dev, i used Phaser.IO,
amazing Javascript 2D Game Engline. In that time i learned
Basic Frontend(HTML, CSS, JS). But it became boring after
some time so i wanted to try something new, and i found a
Web Design(UI/UX). After that, till this day, im trying to be
as better Designer as possible.
